Лабораторна робота № 2

Мова створення сценаріїв JavaScript

Носова Поліна

КІУКІ-19-9 

Варіант 7

Калькулятор, що представляє собою набір кнопок для введення цифр та обрання операції (додавання, віднімання, множення, поділ), поле для виразу, що обчислюється (формується шляхом натискання на відповідні кнопки на сторінці), поле для виведення результату.

У ході виконання лабораторної роботи була створена html сторінка - calculator.html з використанням calclulator.css для стилізації.

Був сторений скірпт calculator.js за допомогою якого вкионуються такі операції як очистка форми, заповнення форми, розраховування результатів.
